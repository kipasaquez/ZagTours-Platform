# 🌏 ZagTours - AI Travel Agent & Multimodal Platform

ZagTours adalah platform Travel Dashboard canggih yang ditenagai oleh kecerdasan buatan (AI) menggunakan teknologi **Google Gemini**. Aplikasi ini dirancang khusus untuk membantu wisatawan merencanakan perjalanan dengan antarmuka yang sangat intuitif dan kaya fitur.

## ✨ Fitur Utama
- **🤖 Asisten AI Multimodal**: Mampu memproses input teks, dokumen, hingga analisis gambar untuk rekomendasi perjalanan instan.
- **🗺️ Smart Itinerary Context**: Panel rincian perjalanan harian yang dinamis di sebelah antarmuka percakapan.
- **⚡ Antarmuka Responsif (Collapsible)**: Layout canggih dimana panel informasi dapat diciutkan secara instan untuk memfokuskan area pandang chat.
- **🥇 Membership & Status Card**: Menampilkan profil perjalanan pengguna, poin loyalitas, dan pintasan trip aktif.
- **💡 Saran Pintar Contextual**: Kartu rekomendasi cepat (kuliner, bahasa, itinerary) yang relevan dengan topik pencarian.

## 🛠️ Stack Teknologi
- **Frontend**: HTML5, CSS3 (Modern Flexible Layouts), Vanilla JavaScript.
- **Backend**: Node.js, Express Framework.
- **AI Engine**: Google Gemini API Core.
- **Libraries**: `marked.js` (Markdown Interpreter), `DOMPurify` (Safe Content).
